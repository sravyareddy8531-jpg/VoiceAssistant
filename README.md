# Voice Assistant using Python

## Description
This is a simple voice assistant project developed using Python.  
The assistant can listen to voice commands and respond using speech.

## Features
- Voice recognition
- Voice response
- Tell current time
- Stop command
- Continuous listening

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- PyAudio

## Commands
- "hello"
- "time"
- "stop"

## How to Run

1. Install required libraries

pip install speechrecognition
pip install pyttsx3
pip install pyaudio

2. Run the project

python assistant.py
